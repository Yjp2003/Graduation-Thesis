# YOLO Vision Dashboard 启动说明

## 环境要求

- Python 3.10+（建议 Conda 环境，含 onnxruntime-gpu）
- Node.js 18+
- NVIDIA GPU + CUDA（可选，无 GPU 自动降级为 CPU）

---

## 启动步骤

### 方式一：一键启动（Windows）

双击运行根目录下的 `start.bat`，自动启动前后端。

### 方式二：手动启动

**1. 启动后端**

```bash
cd d:/YOLO/server
activate yolo_vision
python main.py
```

后端运行在 `http://localhost:3001`

**2. 启动前端**

```bash
cd d:/YOLO
npm run dev
```

前端运行在 `http://localhost:3000`

## 使用流程

1. 打开 `http://localhost:3000`，登录账号
2. 在推理控制区上传 `.onnx` 或 `.pt` 模型（PT 文件会自动转换为 ONNX），或从已保存列表中加载
3. 上传图片或视频文件
4. 用 FPS 滑动选择器选择目标帧率（15 / 30 / 60 / 90 / 150 / 200 / 300）
5. 点击「开始识别」
6. 停止后记录自动保存到数据库，可在历史记录页查看

---

## 可选配置：自动加载模型

在 `server/.env` 中添加：

```env
MODEL_PATH=d:/YOLO/server/models/your_model.onnx
```

后端启动时将自动加载指定模型，无需手动上传。

---

## 历史记录说明

- **图片识别记录**：保存截图缩略图，点击可查看大图及检测详情
- **视频识别记录**：无截图，显示视频图标占位，点击可查看检测详情
